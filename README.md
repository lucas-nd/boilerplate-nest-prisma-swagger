## REST API boilerplate NestJS/Prisma/Swagger

API REST with NestJS Framework, PrismaORM and SwaggerUI for start projects.

## Initial Setup

```bash
$ cp env-example .env
```

## Installation

```bash
$ npm install
OR
$ yarn
```

## Running the app

```bash
# development
$ npm run start
OR
$ yarn start

# watch mode
$ npm run start:dev
OR
yarn start:dev

# production mode
$ npm run start:prod
OR
yarn start:prod
```

## Quick RUN with DOCKER

```bash
$ docker-compose up
OR
$ docker-compose up -d
```
